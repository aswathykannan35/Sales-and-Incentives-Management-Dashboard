# Sales and Incentives Management Dashboard (Power BI)

This Power BI project visualizes sales performance, incentives, and target achievements of sales personnel across different regions.  
The dashboard provides an interactive overview of overall targets, total sales, profit, and individual achievement levels, helping management track performance and incentive distribution effectively.
